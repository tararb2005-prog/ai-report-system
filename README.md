# 🚀 AI Report System

An automated AI-powered report generation system built using Django that analyzes datasets and generates professional PDF reports with insights and visualizations.

---

## 📌 Features

- 📂 Upload CSV dataset
- 📊 Automatic data analysis
- 🤖 AI-generated insights
- 📈 Graph visualization
- 📄 PDF report generation
- 🔐 User login system
- 📁 Dashboard for reports

---

## 🛠 Tech Stack

- Python
- Django
- Pandas
- Matplotlib
- ReportLab

---
